Basics of the web

Above picture shows how a web works. Internet gets request from browser and communicates with servers using HTTP protocol. It then fetches HTML documents from servers and delivers back to browser.

HTML

HTML is a a set of text command that can be used to format how a web page is displayed.
HTML = Hyper Text Markup Language.
It contains: text content, markup , references to other documents and links to other pages.

HTML markup element

A HTML element contains opening tag, content and closing tag.
It looks like this:
<tag name> content < /tag name>
The <tag name> is opening tag.
The content is content.
The < /tag name> is closing tag.
Attributes can be added to opening tag. It looks like this:
<tag name attribute name = " values " > content < /tag name>

Inline vs. blocks

HTML inline tags simply apply styles to the content. Whereas block tags create a invisible box around the content.
Inline tages are:
<b> -- bold tag,
<em> -- italicize tag,
<br> -- line break tag,
<img> -- image tag,
<a> -- link tag,
<span> -- style tag,
<strong> -- define important text,
block tags are:
<p> -- paragraph tag,
<div> -- style tag,
<form> -- create a HTML form for user input,

HTML document structure

<!DOCTYPE HTML> -- this defines document type
<html> -- opening tag of the html document
        <head> -- opening tag of the document head. Document head includes meta-data, java script or CSS style.
                <title> title < /title> -- title of the document
        </head> -- closing tag of the document head
        <body> -- opening tag of the document body. Document body has the content of the docuemnt.
                <b> content < b> -- content of the document. You can use any style tags here.
        </body> -- closing tag of the document body
</html> -- closing tag of the html document
